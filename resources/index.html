Team Resources

Here are some resources that we use to help us with our FIRST® journey. We hope you find them useful too! If you have any questions, please feel free to reach out to us via email.

sotabots@gmail.com

FIRST® Inspires Website

This is where you go for all FIRST® related activities. To sign up for STIMS, volunteer for an event, and retrieve season materials.

2025 Game Manual - REEFSCAPE℠ presented by Haas.

View or download the game manual for the 2025 Season game REEFSCAPE℠.

Team Member Forms

These are forms that we use to help team members and parents volunteer for events, or sign up for private vehicles. More information can be found in the forms themselves. All of these forms are from Tacoma Public Schools and are required to be filled out by all team members and parents/guardians (exc. Private Vehicle Form).

Parent/Guardian Volunteer Form

This is a form that parents/guardians can fill out to volunteer for events or shop hours. This form is used to help us keep track of who is volunteering for what event.

Community Volunteer Form

This is a form that community members NOT enrolled in a Tacoma Public Schools school can fill out to volunteer for events or shop hours. This form is used to help us keep track of who is volunteering for what event.

Private Vehicle Form

This form is for team members and parents/guardians to fill out to sign up for private vehicles. This form is used to help us keep track of how many extra vehicles we can get from team parents or supporters.

Programming Resources

These are resources that we use to help us with our programming. We hope you find them useful too! If you have any questions, please feel free to reach out to us via the contact form on the home page.

WPILIB Documentation

This is the official documentation for WPILIB, the library that we use to program our robot. It contains information on how to use the library, as well as examples and tutorials. WPILIB is also how you start programming your robot. It is a library that is used to program the robot in Java, C++, and even sometimes Python!

Chief Delphi Forums

Chief Delphi is a forum that is used by many FIRST® teams to ask questions and get help with their robot. It is a great place to get help with programming, as well as other topics related to FIRST®. It is a great place to ask questions and get help from other teams. It is also a great place to find out about new tools and resources that are available for FIRST® teams.

NI (National Instruments)

NI is the company that makes the hardware that we use to program our robot. Commonly they are known for their RoboRIO 1.0 and 2.0, which are the brains of the robot. They also make the cRIO, which is used in some older robots. They have a lot of great resources for FIRST® teams, including tutorials and examples. They also have a lot of great resources for programming in LabVIEW, which is a programming language that is used by some FIRST® teams.

FRC Game Tools is a set of tools that are used to help teams with their robot. It includes a lot of great resources for programming, as well as other topics related to FIRST®. It is a great place to ask questions and get help from other teams. It is also a great place to find out about new tools and resources that are available for FIRST® teams.

Limelight Hardware Manager & Documentation

Limelight is a camera that is used to help teams with their robot. It is a great tool for vision processing and can be used to help with autonomous and teleop. It is a great place to ask questions and get help from other teams. It is also a great place to find out about new tools and resources that are available for FIRST® teams.

BalenaEtcher is a tool that our team uses to re-image limelights. Its recommended that you have the limelight directly plugged into your device via USB to re-image it.

REV Documentation

REV Documentation is a great place to find resources for programming and using REV motors, encoders, limit switches, and more.

CTRE (Cross-the-Road Electronics) Documentation & Software

CTRE is a company that supplies the same as REV, they have motor controllers, encoders, and other, the same as REV. They also pair with WCP (West Coast Products) and V3X to provide a lot of great resources for FIRST® teams.

Phoenix Tuner X is a tool by CTRE that works the same as REV Hardware Client, except it is used to configure CTRE motor controllers and other devices.
